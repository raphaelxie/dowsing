# 体用生克 — 失物占断法

## 体用定义

- **体卦**：失主（问卦者本人）
- **用卦**：失物（所找之物 / 走失的宠物）

体用判定规则（与梅花易数通用规则一致）：

- 动爻在第 1–3 爻（下卦）→ **上卦为体，下卦为用**
- 动爻在第 4–6 爻（上卦）→ **下卦为体，上卦为用**

## 能否找回（体用生克）

| 生克关系 | 倾向 | 距离 | 说明 |
|----------|------|------|------|
| 用生体 | 易得 | 近 | 失物自来，多在近处，容易发现 |
| 体用比和 | 易得 | 近 | 同气相求，原处附近即可找到 |
| 体克用 | 可得 | 中 | 需主动寻找，费力但能找回 |
| 用克体 | 难寻 | 远 | 恐已离身，或被他人取走 |
| 体生用 | 难得 | 远 | 耗神费力，多半难以找回 |

> 输出时使用「倾向」「可能」等措辞，不作绝对断言。

## 是否被人拿走

结合变卦与互卦辅助判断：

| 情况 | 倾向解读 |
|------|----------|
| 体克用 + 变卦用卦仍近体 | 物仍在，只是被挪动 |
| 用克体 + 变卦远离体卦五行 | 可能已被他人取走或遗落远处 |
| 互卦克体 | 途中曾受阻，或经他人之手 |
| 变卦与用卦五行相同 | 未离原类场景，仍在相似环境中 |

## 搜寻优先级

1. **方位优先**：用卦后天方位是跨语境最稳定的线索，以**失物最后出现的位置**为原点判断
2. **用卦**类象场景（依语境取象，第一优先搜索区）
3. **变卦**上下两卦合参（若与用卦不同，表示已移动；两卦方位可组合为更精确的复合方向，如离南+兑西=西南）
4. **互卦**上下两卦合参（中间经过处、过渡空间；两半各自提供方向线索）

## 语境（失物所在环境）

失物占不可默认用户在家。起卦前需确认失物语境，以正确取象：

| 语境 | 适用场景 |
|------|----------|
| `home` 居家 | 在家中丢失 |
| `public` 公共场所/户外 | 图书馆、学校、办公室、商场、街道 |
| `transit` 交通工具 | 飞机、大巴、火车、汽车、地铁 |
| `pet` 走失生物 | 走失的猫、狗等宠物 |
| `general` 通用 | 不确定时的默认，侧重方位 |

## 走失生物（宠物）专用

- **用卦方位** = 宠物的去向（以走失地点为原点）
- **内/外卦** = 是否走远（下卦未走远、上卦已走远）
- **体用生克** = 是否自归：
  - 用生体 / 比和 → 宠物倾向自归或在近处，宜呼唤+守候原地
  - 体克用 → 需主动沿方位寻找，可寻回
  - 用克体 / 体生用 → 较难，宜尽快沿方位寻找并发布寻宠信息
- **动物类象（说卦传）**：乾马、坤牛、震龙、巽鸡、坎豕、离雉、艮狗、兑羊，可辅助理解卦象（如艮为狗）

## 断卦步骤（失物占专用）

1. 确认失物语境（居家/公共/交通/宠物）
2. 起卦（时间 / 数字）
3. 定体用（体=失主，用=失物/宠物）
4. **先定方位**（用卦后天方位，以失物最后出现处为原点）
5. 依语境查用卦类象 → 场景
6. 判体用生克 → 能否找回 + 远近 / 是否自归
7. 看变卦 → 是否移动、第二搜索区（**注意变卦上下两卦的复合方向**，引擎已自动计算 `combined_direction`）
8. 看互卦 → 中间路径（**互卦上下两卦各自提供方向线索**）
9. 综合输出【下一步】行动建议

## 伦理

- 吉凶并陈，不作绝对预言
- 不替代报警或专业搜寻（贵重物品遗失可建议报警；公共场所失物建议联系失物招领）
- 强调：本结果是**结构化搜索启发**，用于打破搜寻盲区
